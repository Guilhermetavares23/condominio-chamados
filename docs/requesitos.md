Requisitos Funcionais

- RF01: Criar chamado
- RF02: Atualizar status do chamado
- RF03: Visualizar chamados
- RF04: Filtrar chamados por categoria
- RF05: Registrar histórico de alterações

# Requisitos Não Funcionais

- O sistema deve ser simples de usar
- Deve responder rapidamente às ações do usuário
- Os dados devem ser consistentes e confiáveis

# Regras de Negócio

- Todo chamado deve possuir um status
- Chamados resolvidos não podem ser alterados
- Usuários só podem visualizar seus próprios chamados, exceto administradores
