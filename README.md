Sistema de Gestão de Chamados para Condomínio

Descrição

Sistema desenvolvido para registrar e gerenciar chamados de moradores em condomínios.

 Funcionalidades
	•	Registro de chamados
	•	Classificação por tipo
	•	Acompanhamento de status
	•	Histórico de alterações

 Estrutura do Projeto
	•	docs: documentação
	•	src: dados do sistema
	•	tests: testes
	•	infra: configuração

 Requisitos

Funcionais
	•	Criar chamado
	•	Atualizar status
	•	Visualizar chamados
	•	Filtrar por categoria
	•	Registrar histórico

Não Funcionais
	•	Sistema simples
	•	Resposta rápida
	•	Dados confiáveis

 Regras de Negócio
	•	Todo chamado deve ter status
	•	Chamados resolvidos não podem ser alterados
	•	Usuários só veem seus chamados

 Execução

Projeto acadêmico (não executável).
